#### 网络设备的状态标识。
`UP` 表示网卡处于启动的状态；
`BROADCAST` 表示这个网卡有广播地址，可以发送广播包；
`MULTICAST` 表示网卡可以发送多播包；
`LOWER_UP` 表示 L1 是启动的，也即网线插着呢。
