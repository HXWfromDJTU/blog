# pm2

### 原理

### 启动命令
```
$ pm2 start
$ pm2 restart
```

```
$ pm2 stop
```

```
$ pm2 list
```

### 查看进程启动情况
```
# 查看进程是否存在
$ pm2 ls | grep [进程名 | pid] 

#查看进程具体情况（进程启动，但是进程内可能报错，需要仔细查看下logs信息）
$ pm2 logs [进程名]
```
其他辅助工具
```
#查看进程信息（是否存在）
$ ps aux|grep 

# 通过端口号查PID (怎么查进程名)
$ lsof -I:[端口号] 
```
