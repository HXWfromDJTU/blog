# 字符串的扩展

### 对象上的方法
1️⃣ `codePointAt()`
2️⃣ `includes()`
3️⃣ `startsWith()`
4️⃣ `endsWith()`
5️⃣ `repeat()`  表示将原字符串重复n次
6️⃣ `padStart()``padEnd()` 表示在字符串前后填充其他字符串


### 类静态方法
1️⃣ String.fromCodePoint()  用于从码点返回对应字符
2️⃣ String.raw()  用来充当模板字符串的处理函数，返回一个斜杠都被转义（即斜杠前面再加一个斜杠）的字符串，对应于替换变量后的模板字符串

### 其他作用
1️⃣ 模板字符串
2️⃣ 模板编译
3️⃣ 标签模板
